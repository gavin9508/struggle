## 线程的生命周期
Java 线程在运行的生命周期中的指定时刻只可能处于下面 6 种不同状态的其中一个状态
### 1.NEW
初始状态，线程被构建，但是还没有调用start()方法。
### 2.RUNNABLE
运行状态，java线程将操作系统中的就绪和运行俩种状态称为“运行中”
### 3.BLOCKED
阻塞状态，表示线程阻塞与锁
### 4.WAITING
等待状态，表示线程进入等待状态，进入该状态表示当前线程需要等待其他线程做出一些特定动作
### 5.TIME_WAITING
超时等待状态，该状态不同于WAITING，它是可以在指定时间自行返回的
### 6.TERMINATED
终止状态，表示当前线程已经执行完毕
